# Personal Profile Card

A simple profile card built with pure HTML and CSS as part of Week 1 of my Frontend Web Development internship at Neurofive Solutions. No JavaScript, no frameworks — just semantic HTML and CSS (box model, typography, colors).

## What's inside

- `index.html` – semantic markup (`header`, `section`, `footer`)
- `style.css` – all the styling
- `profile.jpg` – profile photo
- `reflection.txt` – Explanation

## Features

- Profile photo, name, short bio, and 3 social links (GitHub, LinkedIn, Portfolio)
- Fully custom CSS — no Bootstrap/Tailwind
- Responsive from 350px to 500px width
- Hover effects on the social links

## What I learned

Working with `object-fit` and `object-position` to crop the profile photo correctly took a bit of trial and error. I also spent some time getting the `box-shadow` right — wanted the card to have depth without looking too heavy. Making it look good at both small and larger widths meant adjusting padding and max-width a few times.
